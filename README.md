# checkpoint

## Guide to run the application:

### Prerequisites:
- Must have Docker

### Steps: 
1. Download the ZIP file
2. Open Docker in the background
3. Connect to the virtual environment (venv), 
    - If not create one: python -m venv venv
4. Once running, right-click 'docker-development-compose.yml' and select 'Compose Up'
5. Wait until everything is running, the terminal should show whether the local application is running or not
6. Go to a browser and search for 'localhost', the application should be running locally.

### Otherwise:
1. Visit anrheia.com to view fully deployed application.

(Note: Both versions of the application are the same)
